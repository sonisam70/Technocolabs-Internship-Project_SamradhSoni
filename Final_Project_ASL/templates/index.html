ASL Prediction

American Sign Language (ASL) is the primary language used by many deaf
individuals in North America, and it is also used by hard-of-hearing and hearing individuals.
The language is as rich as spoken languages and employs signs made with the hand, along with facial gestures and bodily postures.

Project Tasks

  1. Examine the dataset
  2. One-hot encode the data
  3. Define the model
  4. Compile the model
  5. Train the model
  6. Test the model
  7. Visualize mistakes
